# MorixOS Timer

## The timer allows you to:

Set a duration in seconds

See a live countdown in the console

Restart the timer without leaving MorixOS

# How to Use

Inside MorixOS, use the following command:

morix/timer

This will open the built-in timer application.

## Behavior

The timer runs in the console

Time is displayed continuously while counting down

When the time reaches zero, a message is shown

You can decide whether to start another timer or return

## Notes

Only numeric input is supported

The timer is part of the MorixOS core system

Designed to be simple and lightweight
